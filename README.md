# My First Program - Calculator

This is a simple Python calculator project.

## Requirements

- Python 3.14+  
  Download: https://www.python.org/ftp/python/3.14.6/python-3.14.6-amd64.exe

## Installation

Open Command Prompt (CMD) and install the required package:

```bash
pip install pyfiglet
```
